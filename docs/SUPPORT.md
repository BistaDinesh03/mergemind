\## Bug Reports



Found a bug? Please open an issue:



1\. Go to \[Issues](https://github.com/BistaDinesh03/mergemind/issues)

2\. Click \*\*New Issue\*\* → \*\*Bug Report\*\*

3\. Include:

&#x20;  - What you were doing

&#x20;  - What you expected to happen

&#x20;  - What actually happened

&#x20;  - Screenshots (if applicable)

&#x20;  - Browser and version

&#x20;  - Steps to reproduce

\## Common Issues



\### Can't sign in

\- Make sure you're using the correct URL: \[https://mergemind-tau.vercel.app](https://mergemind-tau.vercel.app)

\- Clear browser cookies and try again

\- Check GitHub OAuth app permissions



\### Dashboard shows 0 repos

\- Ensure you've signed in with GitHub

\- Check that you have public repositories

\- Try refreshing the page



\### Repository "Not Found" error

\- The repository may have been deleted or renamed on GitHub

\- Try searching for the repository on the Discover page



\### Search not working

\- Type at least 2 characters

\- Check your internet connection

\- The backend API may be temporarily unavailable
