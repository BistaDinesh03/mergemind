# Changelog

All notable changes to MergeMind are documented in this file.

## [1.0.0] - 2026-07-05

### Added
- GitHub OAuth login via NextAuth
- Repository health analysis (activity, documentation, community, maintenance)
- Issue scoring engine (difficulty, merge probability, time, beginner friendliness, repo health, clarity)
- AI Mentor explanations for every recommendation
- Portfolio tracking from merged pull requests
- Dashboard with ranked issue recommendations
- Discover page for browsing and filtering repositories
- Repository analysis view with health metrics
- Issue analysis view with per-issue scoring breakdown
- Dark mode throughout the interface
- Docker and Docker Compose setup
- GitHub Actions CI workflow
- Command palette (Cmd+K)
- WCAG 2.2 AA accessibility
- 16 backend tests (pytest)

### Changed
- Default local database uses SQLite for development persistence
